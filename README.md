A comprehensive, interactive dashboard to track and visualize COVID-19 vaccination progress worldwide. Built with **Streamlit**, **Pandas**, and **Plotly**.

## ğŸš€ Features

- **Global Overview**: Track total vaccinations, people fully vaccinated, and booster doses worldwide.
- **Interactive Visualizations**:
  - **Dynamic Trends**: Interactive histograms and line charts showing daily and cumulative vaccination progress.
  - **Global Map**: Choropleth map visualizing vaccination coverage by country.
  - **Comparative Analysis**: Compare "People Vaccinated" vs. "Fully Vaccinated" for any country.
  - **Top Rankings**: View the top 10 countries by total vaccinations.
- **Data Optimization**: Uses advanced data cleaning (forward filling) and **Parquet** format for high-performance data loading.
- **User-Friendly Interface**:
  - Date Range Filtering.
  - Dedicated tabs for Trends, Comparisons, and Global Views.
  - Responsive layout.

## âš™ï¸ Setup & Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/covid-dashboard.git
   cd covid-dashboard
   ```

2. **Create a virtual environment (Optional but recommended)**
   ```bash
   python -m venv venv
   # On Windows
   venv\Scripts\activate
   # On macOS/Linux
   source venv/bin/activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Prepare the Data**
   Run the Jupyter notebook `covid_dashboard.ipynb` to download the latest data from *Our World in Data*, clean it, and generate the `cleaned_vaccinations.parquet` file.
   
   *Alternatively, if the parquet file is already included, you can skip this step.*

5. **Run the App**
   ```bash
   streamlit run app.py
   ```
